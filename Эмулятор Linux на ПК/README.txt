Guake — это выпадающий терминал для настольных компьютеров на базе gtk, таких как gnome, xfce и т.д. Это полезный инструмент, который вызывает терминал при нажатии клавиши. Guake написан на Python. Он поддерживает горячие клавиши, вкладки, прозрачный фон и многое другое. 

Мануал по установке:

Установка Guake

Первым шагом будет установка guake. Он доступен в репозиториях, поэтому установим его сразу:

$ sudo apt-get install guake

 
Исправляем ошибку — «Guake не может быть инициализирован!»

После установки попробуйте запустить Guake из меню приложения. Он должен быть в категории «Система». На Xubuntu Guake может не запускаться со следующим сообщением об ошибке:

Guake can not init!
Gconf Error.
Have you installed guake.schemas properly?

 

Вышеприведенная ошибка возникает, поскольку файл guake.schemas не найден в необходимом расположении. Чтобы исправить эту ошибку, просто запустите следующие команды:

$sudo mkdir /etc/gconf/schemas
$cd /etc/gconf/schemas/
$sudo ln -s /usr/share/gconf/schemas/guake.schemas

 

Попробуйте снова запустить Guake, теперь он должен заработать. При нажатии F12 появляется всплывающий терминал.

Настройка Guake для запуска при входе в систему

В Ubuntu guake автоматически не запускается при входе в систему. Это нужно делать вручную, но исправить это не сложно.

Откройте «All settings» => «Session and Startup». Перейдите на вкладку «Application Autostart» и нажмите «Add». Введите следующие данные:

Name - Guake
Description - Guake dropdown terminal
Command - /usr/bin/guake

 

Нажмите «ОК» и закройте настройку. В следующий раз, когда вы запустите ПК, терминал Guake должен загрузится сразу после входа на рабочий стол.

Вы также можете запустить следующую команду для настройки guake для автозапуска при входе в систему.

$cp /usr/share/applications/guake.desktop /etc/xdg/autostart/

 

Теперь вы знаете, как работать с Guake и вызывать выпадающий терминал!
