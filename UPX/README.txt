UPX - это упаковщик исполняемых (.exe) файлов. Упаковка помогает существенно уменьшить размер файла, что может помочь при передачи по сети или на носителе. Для сжатия файлов, UPX использует алгоритм "сжатия без потерь" UCL. Большой плюс программы - открытый исходных код и поддержка многих форматов файлов. У программы нет интерфейса, все действия из консоли.
